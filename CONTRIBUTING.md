# Contributing to RPStories 🤝✨

Thank you for your interest in **RPStories**! We are building the next generation of immersive character management for the Roleplay community. Our goal is to move from a static archive to a full-featured web application where users can manage their own private "safe space" of characters and share them seamlessly for whitelists and lore-tracking.

## 🚀 Our Vision
We want to provide a premium, game-inspired interface where players can:
- Create and manage multiple character dossiers.
- Keep private notes and backgrounds in a secure "Safe".
- Generate shareable links for server applications (Whitelists).
- Explore a global, interconnected genealogy hub.

## 🌟 How can you help?

Since we are evolving towards a full web app, we need help in the following areas:

### 1. Application Development (Vue 3 / Vite)
Help us build the core features of the platform:
- User authentication and profile management.
- The "Character Creator" interface (moving from code-based templates to a real UI).
- Database integration for persistent storage.

### 2. UI/UX Design (Tailwind CSS v4)
Our aesthetic is "Premium & Immersive". We need contributors to:
- Refine existing components for a more "game-like" feel.
- Design the new dashboard and creation workflows.
- Improve mobile responsiveness and accessibility.

### 3. Feature Ideas & Feedback
Tell us what features would make your RP life easier! Whether it's a specific type of dossier or a new way to visualize relationships, your feedback is invaluable.

## 🛠️ How to Submit Changes

1. **Fork** the repository.
2. **Create a branch** for your feature or fix: `git checkout -b feature/amazing-feature`.
3. **Commit** your changes: `git commit -m 'Add some amazing feature'`.
4. **Push** to the branch: `git push origin feature/amazing-feature`.
5. **Open a Pull Request**!
